在Mac上使用Markdown

导语

MarkDown是一种轻量级的标记语言，他的开发者为John Gruber。通过简单的标记语法，它可以使普通文本具有一定的格式

​ 最近入手一台macbook pro，已经把这个博客从windows迁移到mac上来了！！！

一、常用语法及快捷键

1.1标题

##一级标题

####二级标题

######三级标题

########四级标题

##########五级标题

############六级标题

快捷键为cmd+X（X为几级标题就写数字几，如二级标题为：cmd+2）

1.2引用

当你需要引用别人的话时，可以在引用的文字前面添加引用标记 >，
即表示此后的内容为引用，这时候，文字的背景色也会改变。

如：>这是一段引用

1.3链接

链接分为图片链接和网址链接。

图片链接

![图片描述] (图片地址)

快捷键为：shift + cmd + I

网页链接

[网页描述](网页地址)

快捷键为 sihft + cmd + k

1.4代码

使用两个 `符号把单行代码包裹起来。

多行代码两个用`包起来。

1.5表格

| 1 | 2 | 3 |

| —- |:—-:| —:|

| a | d | g |

| b | e | h |

| c | f | I |

| —- |:—-:| —:|表示对其方式。:—表示左对齐；—:表示右对齐；:—-:表示中间对齐。

效果为：

1 2 3
a d g
b e h
c f I
1.6其他
加粗 :**Bold**
斜体字 :*Italics*
高亮 :==text==
段落 : 段落之间空一行
画水平线 (HR) :--------
方框：- [ ] -
